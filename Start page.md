obsidian использует utf-8 для кодировки содержимого файлов. Однако кодировкой имён этих файлов занимается операционная система, которая может использовать ,например, cp-1251, от чего будут возникать проблемы с интерпретацией символов. Проблемы были обнаружены уже в git. Так что лучше названия файлов задавать с использованием латинских символов

Для формирования автоматических заголовков использовать [это расширение](obsidian://show-plugin?id=automatic-table-of-contents)
