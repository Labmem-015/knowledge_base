# One-hot encoding
Мы кодируем категориальные данные в вектор-столбец, размером в количество категорий. Единица ставится в том элементе, которая соответствует категории, которую мы хотим закодировать. Остальные элементы заполняются нулями.

# Скалярное произведение
В случае с one-hot закодированными словами, мы можем сравнивать их: если равны, то результат один, если не равны, то ноль. Также, если мы имеем дело с закодированным набором слов, где элементы представляют собой веса, то чтобы узнать силу влияния одного конкретного слова, то мы можем использовать скалярное произведение с one-hot закодированным вектором, тем самым вычленяя значение веса интересуемого нас слова.

Если нам нужны конкретно эти операции, то имеет смысл оптимизировать данные операции, используя только индексы массивов, без использования всего массива в целом. Но это только в случае, если мы работает с одномерными массивами данных и только если он представлен в one-hot encoding
