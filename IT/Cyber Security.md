# Статический базовый анализ
* Анализ сигнатур при помощи антивируса
* Вычисление хэша (MD5, SHA-1)
* Поиск строк при помощи утилит (Strings из Sysinternals на WIndows, или strings на UNIX). Там можно задать критерии поиска (например длину строк от `n` символов). Можно искать как ASCII, так и Unicode последовательности (на Windows это wstring, т.е. UTF-16).
