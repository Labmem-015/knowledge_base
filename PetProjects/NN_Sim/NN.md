С точки зрения отображения памяти общая система состоит из слоёв, каждый слой состоит из весовых коэффициентов, который соединяют его с предыдущим слоем.

Таким образом у нас формируется двумерный массив данных.

# Динамическая архитектура
Нужно создать функцию, которая должна будет решать, создавать ли новые блоки нейросетей, например с помощью механизма attention в Google Titans. Нужно убедится, что мы должны создавать новый блок памяти, а не прибегать к дообучению существующего. Также возможно использовать Z-вектора для оценки размеров новых блоков, а не только для указания, какую область нейросети задействовать. Или эти Z-вектора будут определять создание различных областей знаний внутри модели. То есть создание блока, отвечающего за матан, создание блока, отвечающего за абстрактную речь и т.д. Можно создать вспомогательную нейронку которая будет отвечать за подбор параметров для своей эволюции.
