С точки зрения отображения памяти общая система состоит из слоёв, каждый слой состоит из весовых коэффициентов, который соединяют его с предыдущим слоем.

Таким образом у нас формируется двумерный массив данных.

# Динамическая архитектура
Нужно создать функцию, которая должна будет решать, создавать ли новые блоки нейросетей. Например с помощью механизма attention в Google Titans, а также может использовать Z-вектора для оценки размеров новых блоков, а не только для указания, какую область нейросети задействовать. Можно создать вспомогательную нейронку которая будет отвечать за подбор параметров для своей эволюции.
